# Front-end Portfolio

## 프로젝트명: Pro-watcha

### ㅇ프로젝트 개요 
- 현재 운영중인 '왓챠플레이어' 를 기반하여 ReactJS를 사용하여 html,css,js 직접 제작하여 만들었습니다.  

### ㅇ스킬
- React
- Redux
- axios
- Styled-Component

### ㅇ기능
- 이미지 슬라이더를 JS로 직접 제작하여 슬라이드 효과를 주었습니다.
- MovieDB Api를 를 사용하여 매주 업데이트 되는 영화를 장르별로 볼 수 있습니다.
- 영화를 클릭하면 모달창이 떠서 영화에 대한 상세한 정보를 볼 수 있습니다.
